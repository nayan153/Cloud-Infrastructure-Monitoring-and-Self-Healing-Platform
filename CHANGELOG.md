# Changelog

All notable changes to this project are documented in this file.

---

## [v1.0.0] - Initial Release

### Added

- Created an Ubuntu EC2 instance on AWS.
- Configured Security Groups and SSH access.
- Developed Bash scripts for CPU, memory, disk, and service monitoring.
- Added an Alert Manager to detect high resource usage.
- Implemented a basic Self-Healing mechanism for selected services.
- Automated monitoring using Cron jobs.
- Added log backup functionality.
- Installed and configured Node Exporter.
- Installed and configured Prometheus.
- Installed and configured Grafana.
- Created monitoring dashboards.
- Configured an IAM Role for secure AWS access.
- Installed and configured the Amazon CloudWatch Agent.
- Created a CloudWatch Dashboard.
- Configured CloudWatch Alarms.
- Integrated Amazon SNS for email notifications.
- Added deployment scripts for easier setup.
- Added architecture diagrams and project documentation.
- Organized the repository with screenshots, reports, and configuration files.
