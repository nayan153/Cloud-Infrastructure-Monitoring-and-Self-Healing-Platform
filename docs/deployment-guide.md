# Server Preparation

## Overview

After creating the EC2 instance, I prepared the Ubuntu server for the project deployment.

This step included updating the operating system, installing the required packages, and creating the project workspace.

---

## Packages Installed

The following packages were installed before starting the project:

* Git
* Wget
* Curl
* Unzip
* Vim
* Tree

These tools are used throughout the project for downloading software, managing files, and working with GitHub.

---

## Project Directory

The project directory was created inside the home directory of the Ubuntu server.

```text
/home/ubuntu/Cloud-Infrastructure-Monitoring-and-Self-Healing-Platform
```

This directory contains all project files, monitoring scripts, configuration files, and documentation.

---

## What I Completed

* Connected to the EC2 instance using SSH.
* Updated the Ubuntu system.
* Installed the required packages.
* Created the project directory.
* Verified that the server was ready for deployment.

---

## Result

The server is now prepared for installing the monitoring platform and related services.
